## ğŸ¨ BAGIAN 2: PANDUAN UNTUK CONTENT CREATOR

### [ğŸš€ Panduan Lengkap untuk Content Creator](User-Guide-Untuk-Creator.md)
â±ï¸ **Estimasi Baca: 30 menit**

**Topik yang Dibahas:**

### **A. Memulai (Getting Started)**
1. **Pendaftaran**
   - Syarat dan ketentuan creator
   - Mengisi form pendaftaran
   - Verifikasi email
   - Memilih username

2. **Setup Profil**
   - Upload foto profil
   - Menulis bio menarik
   - Menambahkan link sosial media
   - Customize halaman profil

3. **Verifikasi Akun**
   - KYC (Know Your Customer)
   - Upload dokumen identitas
   - Proses review (1-3 hari)

### **B. Menerima Donasi**
4. **Share Link Profil**
   - Copy link profil
   - Share ke YouTube, Instagram, Twitter
   - Buat QR Code untuk offline event
   - Pasang link di bio

5. **Notifikasi Donasi**
   - Menerima email notifikasi
   - Real-time alert di dashboard
   - Setup notifikasi browser
   - Integration dengan Discord/Telegram (opsional)

6. **Melihat Dashboard**
   - Total earnings
   - Grafik donasi per bulan
   - Top supporters
   - Recent donations

### **C. Pencairan Dana (Payout)**
7. **Syarat Payout**
   - Minimum saldo: Rp 50.000
   - Verifikasi rekening bank
   - Tidak ada payout pending

8. **Request Payout**
   - Masuk ke halaman Payout
   - Input jumlah dan rekening tujuan
   - Submit request
   - Menunggu approval admin

9. **Status Payout**
   - PENDING: Menunggu review admin
   - APPROVED: Disetujui, akan diproses
   - PROCESSED: Uang sudah ditransfer
   - REJECTED: Ditolak (dengan alasan)

10. **Tracking Payout**
    - Cek status di halaman Payout History
    - Notifikasi email setiap perubahan status
    - Estimasi waktu transfer

### **D. Tips Sukses**
11. **Memaksimalkan Donasi**
    - Konsisten create content
    - Engage dengan audience
    - Call-to-action di video/post
    - Buat konten eksklusif untuk supporters

12. **Etika & Aturan**
    - Larangan meminta donasi paksa
    - Transparansi penggunaan dana
    - Respect kepada supporters
    - Aturan platform yang harus diikuti

13. **Troubleshooting Creator**
    - Lupa password
    - Donasi tidak masuk
    - Payout ditolak
    - Akun suspended

14. **FAQ Content Creator**
