## ğŸš€ Quick Start

### Sebagai Donatur:
1. Kunjungi profil creator favorit: `nyumbangin.com/[username]`
2. Klik tombol **"Donasi Sekarang"**
3. Isi jumlah dan pesan (opsional)
4. Pilih metode pembayaran
5. Selesaikan pembayaran

### Sebagai Creator:
1. Daftar di `nyumbangin.com/daftar`
2. Verifikasi email Anda
3. Lengkapi profil (foto, bio, sosial media)
4. Share link profil Anda ke audience
5. Terima donasi dan request payout

### Sebagai Admin:
1. Login di `nyumbangin.com/admin`
2. Gunakan kredensial admin yang diberikan
3. Akses dashboard untuk monitoring
4. Review dan approve payout requests
